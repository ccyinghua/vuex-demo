# vuex-demo
vuex学习-集中式管理数据

### 运行
```javascript
npm install
npm run dev
```
- src 文件夹--复杂vuex
- src1 文件夹--简易vuex

说明：默认运行src,若要运行src1，文件夹名字更换即可
